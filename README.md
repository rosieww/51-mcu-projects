# 51 MCU 入門實作專案

這是一個包含 51 MCU (8051）各項功能練習的程式碼集合。用來熟悉 51 單晶片的硬體控制與程式設計，涵蓋 GPIO、計時器、中斷、UART、通信 等基本模組的應用。

---

### 專案特色

* **模組化程式碼**：每個練習項目都獨立成一個資料夾，方便學習和管理。
* **逐步學習**：從最簡單的 LED 閃爍開始，逐步深入到更複雜的模組應用。
